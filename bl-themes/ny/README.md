# NY theme for Bludit
NY is a simple and minimalistic theme based on [Moran](https://github.com/echenley/moran) theme for Nyssen. 

## Features
- Cover image support
- Fully responsive
- Footer with social links
- Static pages are automatically added to the navbar

## Compatible
- Bludit v3.x

## Author
- Nyssen.
